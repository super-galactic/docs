Super Galactic smart contracts have undergone independent third party security review covering the core on-chain systems used by UAP, NFTs, and the Super Galactic Hub.

Super Galactic smart contracts were audited by [CertiK](https://skynet.certik.com/projects/ufo-gaming?auditId=UFO#code-security).

### Audit Scope

The audit covered the core smart contracts powering the Super Galactic ecosystem, including:

- UAP token contracts
- Genesis and asset NFT contracts
- Marketplace and Super Galactic Hub contracts
- Core ownership and transaction logic

The review focused on identifying vulnerabilities, logical flaws, and potential attack vectors within the contract codebase.

### Audit Status

- Audit completed
- No critical unresolved issues reported
- All identified findings reviewed and addressed where required

The audited contracts are currently deployed and used within the live Super Galactic ecosystem.
