This page covers how to access Super Galactic, select a platform, and enter gameplay.

### 1. Choose a Platform

Super Galactic is available on browser, Windows, and Android. Platform choice does not change gameplay rules, progression eligibility, or reward eligibility.

- Browser: https://sg.ufogaming.io
- Windows: https://storage.ufogaming.io/ufogaming-game-cdn/supergalactic_launcher/production/SupergalacticWin64Launcher-v0.1.0.exe
- Android: https://play.google.com/store/apps/details?id=com.UFOGaming.SupergalacticGame

### 2. Select Controls and Input

Supported input methods:

- Mouse and keyboard
- Xbox compatible controllers

Input selection does not affect matchmaking or progression rules.

### 3. Select a Server Region

Select a server region before entering gameplay. Use the same region as teammates to play together.

For region rules and matchmaking behaviour, see Gameplay → [Matchmaking & Servers](/game/gameplay/Matchmaking-Servers).

### 4. Enter Gameplay

Start in Demo Mode to learn movement, aiming, weapons, and abilities. Progression and Play to Earn access depends on wallet connection and Genesis ownership.

For requirements, see Gameplay → [Access & Requirements.](/game/gameplay/access-and-requirements)

### 5. Optional Tutorials

Tutorials are available for players who want a guided introduction to core mechanics. Tutorials can be completed before entering live gameplay.
